## 变量是什么

计算机用来存储数据的“容器”


<Alert type="info">
变量不是数据本身，他们仅仅是一个存储数据的盒子
</Alert>

## 变量基本使用

### 声明变量

```
//语法： let 变量名
let age
```

### 变量初始化

```
let age = 18
// = 赋值
```

### 声明多个变量(建议分开声明)

```

```
